# FitPlanner

Aplicativo web estático para montar planos iniciais de treinos para emagrecimento e dietas personalizadas com base no objetivo, peso, altura e rotina do usuário.

## Funcionalidades

- cálculo estimado de IMC, gasto energético diário e meta calórica;
- divisão inicial de macronutrientes;
- sugestões de refeições para o dia;
- cronograma semanal de treinos conforme os dias disponíveis;
- personalização por objetivo e preferências alimentares.

## Como usar

1. Abra o arquivo `index.html` no navegador; ou
2. rode um servidor local simples, por exemplo:

```bash
python3 -m http.server 8000
```

Depois acesse `http://localhost:8000`.

## Observação

O app fornece apenas uma estimativa educacional inicial e não substitui acompanhamento profissional.
